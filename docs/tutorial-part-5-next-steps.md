# Part 5: Next Steps

Congratulations! You've designed a complete ergonomic keyboard in Ergogen. You have a layout, an outline, a PCB, and a case design. Now what?

The final steps of building your keyboard happen outside of Ergogen. This guide will give you an overview of what you need to do next and point you to some useful resources.

## 1. Routing in KiCad

The PCB file generated by Ergogen is **un-routed**. This means that while all the components and their connections (nets) are defined, the physical wires (traces) that connect them have not been drawn yet. Your next step is to open the `.kicad_pcb` file in [KiCad](https://www.kicad.org/) and route the traces.

Routing can be a complex process, but for a keyboard, it's a manageable task. Here are some resources to help you learn KiCad:

*   **[Creating a PCB in Everything: KiCad](https://hackaday.com/2016/12/23/creating-a-pcb-in-everything-kicad-part-3/)**: A great tutorial series that covers the basics of KiCad.
*   **[KiCad's Official Getting Started Guide](https://docs.kicad.org/#_getting_started)**: The official documentation from the KiCad team.
*   **[Ben Vallack's Keyboard-Specific KiCad Video](https://youtu.be/M_VuXVErD6E)**: A video tutorial focused on routing a keyboard PCB.

## 2. Fabricating the PCB

Once you've routed your PCB in KiCad, you're ready to send it to a manufacturer to be fabricated. You'll need to export your design as a set of Gerber and drill files.

In KiCad, you can do this from the `File -> Fabrication Outputs` menu. Make sure to generate both Gerber files and drill files, and then zip them up into a single file for the manufacturer.

Some popular PCB fabrication services for hobbyists include:

*   **[JLCPCB](https://jlcpcb.com/?from=ERGOGEN)**
*   **[PCBWay](https://www.pcbway.com/)** (use coupon code `ERGOGEN` for a discount)
*   **[OSH Park](https://oshpark.com/)**

## 3. Sourcing Components

While your PCBs are being made, you can order the rest of your components. Here's a sample Bill of Materials (BOM) for the keyboard we designed in this tutorial:

| Part | Amount |
| :--- | :--- |
| PCB | 1 (but you'll likely get 3-5 from the fab) |
| Arduino Pro Micro (or compatible) | 1 |
| OLED Display (128x32) | 1 |
| Kailh Choc v1/v2 Switches | 58 |
| 1N4148 Diodes (SMD or through-hole) | 58 |
| Kailh Choc Hotswap Sockets | 58 |
| Reset Switch (6mm, 4-pin) | 1 |
| Sockets and Pins for Microcontroller | 1 set |
| Choc v1 Keycaps | 1 set (with enough 1u and 1.5u keys) |
| M2 Screws and Threaded Inserts (for the case) | 8 of each |

You can find these parts at various online stores, such as [42Keebs](https://42keebs.eu/), [Little Keyboards](https://www.littlekeyboards.com/), and [Boardsource](https://boardsource.xyz/).

## 4. Firmware

Your keyboard needs firmware to run. The two most popular open-source firmwares for custom keyboards are [QMK](https://qmk.fm/) and [ZMK](https://zmk.dev/).

*   **[QMK (Quantum Mechanical Keyboard)](https://docs.qmk.fm/#/):** The most established and feature-rich firmware. It's great for wired keyboards like the one we designed.
*   **[ZMK (Z-MK)](https://zmk.dev/docs):** A newer firmware designed with wireless keyboards in mind, but it also works for wired keyboards.

Both firmwares have excellent documentation that will guide you through the process of creating a configuration for your new keyboard. You'll need to tell the firmware which pins you used for your rows and columns, and define your keymap.

## 5. Assembly

Once you have your PCBs and all your components, it's time to assemble your keyboard! This will involve soldering the diodes, hotswap sockets, reset switch, and microcontroller headers to the PCB.

If this is your first time soldering, there are many great tutorials on YouTube. It's a good idea to practice on a cheap soldering kit before working on your keyboard.

For a keyboard similar to the one we designed, the [Sofle Choc build guide](https://choc.brianlow.com/pages/build) is an excellent resource.

## Conclusion

Building a custom keyboard from scratch is a rewarding experience. You've learned how to design a keyboard layout, create a PCB, and generate a case using Ergogen. The next steps will take you into the world of electronics and firmware, but with the resources provided here, you're well on your way to typing on a keyboard that you designed and built yourself.

Happy building!
